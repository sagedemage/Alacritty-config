# alacritty config
Configuration for fast terminal emulator called alacritty.

## alacritty config location (linux)
```
~/.config/alacritty/alacritty.yml
```

# alacritty desktop file location
```
$HOME/.local/share/applications
```
``
$HOME=/home/user
``
